# Glad You're Here!

Level 1 consists of a dozen hidden eggs through the site.

They don't have to be found in any certain order but they
are hidden in increasingly difficult ways.

- Storage
- Implementation
- Certain actions, places, etc...
- Passwords
- Puzzles
- Mini Games
